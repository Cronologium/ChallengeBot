Description

X and O is a game which is played out of boredom on a piece of paper with a 3 X 3 grid. Each player has its own marker, X or O. The player with X gets to start. Players take turns in placing their marker in one of the grid's squares. The goal of the game is for one player to obtain a full line, a full column or a full diagonal with their marker.

Code description

Your program must implement a single function, named 'put', inside the class 'Solution' which will return a square to put your mark on.

Return type

Your function will as a string, the coordinates of the square, as such: "x y".

Api description

You are provided with an API object as parameter. The object has two fields available.

Name Role Type
api.board Stores the status of the board char[][]
api.playing The mark ('X' or 'O') you are playing with char

Each square of api.board can take the values 'X', ' ', 'O' (This is a big letter o, not a zero!)

Specifications

  • The memory limit is applied to the whole program
  • Before challenging others, you program must be validated
