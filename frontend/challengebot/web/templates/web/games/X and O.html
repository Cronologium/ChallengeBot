Description

X and O is a game which is played out of boredom on a piece of paper with a 3 X 3 grid. Each player has its own marker, X or O. The player with X gets to start. Players take turns in placing their marker in one of the grid's squares. The goal of the game is for one player to obtain a full line, a full column or a full diagonal with their marker.

Code description

You must implement a program which, at the begining, gets the playing mark. If playing with X, prints the coordinates of where X should be put, then reads the position of an O put by the opponent. If playing with O, your program reads the coordinates of an X put by the opponent then prints the coordinates of an O.

Specifications

  • The memory limit is applied to the whole program
  • The time limit is applied for everything that happens in between reads and writes.
  • Before challenging others, you program must be validated
