Welcome to Challenge Bot!

What?

This is a platform for gamers and coders (at the same time) to test their AI skills and game strategies. It is the perfect combination between coding and playing. The idea is simple: you read the game rules, implement a solution for different game situation, which as whole, make the AI of a game. You submit it and then you can challenge other people!

How?

  • Register using the upper-right button
  • Log in
  • Choose a game
  • Submit your code
  • Play with others by challengeing them!

What's next?

Our simplest game is Battleships. We recommend you check out that game and submit your solution!

Supported languages

    We have support for the following:
  • Python 2.7
  • C++
